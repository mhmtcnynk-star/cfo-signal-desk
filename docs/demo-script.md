# Suggested Demo Video Script

## 0:00 - Opening

"This is CFO Signal Desk, an AI-powered cockpit for finance leaders. It answers the daily executive question: What happened today, why does it matter, and what should I do tomorrow?"

## 0:15 - Problem

"CFOs and FP&A teams are flooded by macro headlines, FX moves, inflation signals, supplier pressure, liquidity constraints, and internal priorities. The issue is not lack of information. The issue is turning fragmented signals into decisions fast."

## 0:35 - Dashboard

"The dashboard starts with Today's Brief, then separates risks, opportunities, FX, inflation, interest rates, company priorities, a watchlist, and action items."

## 0:55 - Risk Detection

"Each signal is classified by severity. Here the FX move is high severity because it directly affects import purchasing, supplier payments, and treasury exposure."

## 1:15 - Priorities

"A CFO can select company priorities such as Cash Flow, Working Capital, FX Exposure, Procurement, Treasury, or Cost Optimization. These priorities steer the AI output."

## 1:35 - AI Brief

"When I generate the brief, CFO Signal Desk returns a board-ready structure: executive summary, key signals, financial impact, operational impact, recommended decisions, and tomorrow's watchlist."

## 2:05 - Decision Value

"The key is that the product does not stop at summarization. It converts signals into actions: review import schedules, accelerate supplier payments if FX risk is higher than financing cost, refresh margins, and prioritize collections."

## 2:30 - Reliability

"The MVP includes demo mode, so it always works without external market APIs. If an OpenAI API key is configured, it uses the OpenAI Responses API for live AI-generated briefings."

## 2:50 - Close

"CFO Signal Desk gives finance leaders a five-minute operating brief for tomorrow's decisions. It is built for practical usefulness: signal over noise, board-ready output, and immediate finance actions."
